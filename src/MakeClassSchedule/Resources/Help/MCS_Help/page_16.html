Room
    The Room class has an ID and the name of the classroom, as well as the number of seats (size of room) and information about equipment (computers or other types). For example if the classroom has computers, it is expected that there is a computer for each seat. IDs are generated internally and automatically.
     
     
    If you want to use a room by specific of room types, So first enter that room types and save it then add your room data with select saved room type.
